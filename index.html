About Convo IDE

Welcome to Convo IDE, a lightweight editor designed to work with ConvoHTML. ConvoHTML is a simplified markup language that lets you create web pages with an intuitive syntax.

Basic Functions:

  • Write ConvoHTML: Use the textarea to input your ConvoHTML code.
  • Preview: Click the 'Preview' button to see how your ConvoHTML code renders as HTML.
  • Save: Click the 'Save' button to download your ConvoHTML code as an HTML file.
  • Download Guidelines: Click the 'Download Guidelines' button to get a PDF guide on how to use ConvoHTML effectively.
